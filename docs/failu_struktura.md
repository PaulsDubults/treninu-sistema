# Failu un mapju struktūras apraksts

- `app/` - Flask lietotnes servera puses kods
- `templates/` - HTML veidnes
- `static/` - CSS un JavaScript faili
- `tests/` - testēšanas faili
- `docs/` - dokumentācija, ER modelis, prezentācijas saturs un pielikuma faili
- `instance/` - lokālā SQLite datubāze
- `run.py` - Flask palaišanas ieejas fails
- `seed_data.py` - datubāzes un testa datu izveides fails
- `requirements.txt` - izmantoto Python bibliotēku saraksts
- `README.md` - projekta apraksts un palaišanas instrukcija
- `.gitignore` - faili un mapes, kuras nevajag likt GitHub
